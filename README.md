# 🔐 ForensicCracker v2.0
### AI/ML-Enhanced Password Recovery & Cyber Forensics Toolkit (Web-Based)

> ⚠️ This tool is intended strictly for **authorized forensic investigations**, **academic research**, and **ethical security testing**.

---

## 🚀 Overview

ForensicCracker v2.0 is an advanced **cyber forensics and password recovery platform** that combines:

- Machine Learning (Adaptive attack selection)
- OSINT-based intelligence gathering
- Multiple password cracking techniques
- Forensic-grade logging and reporting

It provides a **web-based interface** for investigators to analyze, crack, and document digital evidence efficiently.

---

## ✨ Key Features

### 🔑 Password Cracking Engine
- Dictionary Attack
- Brute Force Attack
- Hybrid Attack
- AI-Optimized Attack Selection (Multi-Armed Bandit - UCB1)

### 🧠 AI/ML Integration
- Learns best attack strategies over time
- Tracks success rates of different techniques
- Dynamically improves cracking efficiency

### 🌐 OSINT Wordlist Generator
- Generates custom wordlists from:
  - Names
  - Dates
  - Personal info
- Enhances success rate of targeted attacks

### 📂 Evidence Processing
- Supports:
  - `/etc/shadow`
  - `pwdump`
  - SQLite databases
  - CSV files

### 📊 Batch Processing
- Crack multiple hashes automatically
- Efficient for large-scale investigations

### 🔍 Hash Identification
- Detects hash type automatically
- Supports multiple formats

### 📜 Forensic Logging
- Chain-of-custody logs
- HMAC-signed logs (tamper-proof)
- Timestamped case records

### 📄 Report Generation
- Export professional forensic reports (PDF)
- Includes:
  - Attack details
  - Results
  - Evidence summary
